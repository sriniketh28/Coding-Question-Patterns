### Problem Statement 
#### We are given an array containing ‘n’ numbers from 1 to ‘n’ 
#### Write a function to sort the numbers in-place and without any extra space.

#### Example 1:
```
Input: [3, 1, 5, 4, 2]
Output: [1, 2, 3, 4, 5]
```
#### Example 2:
```
Input: [2, 6, 4, 3, 1, 5]
Output: [1, 2, 3, 4, 5, 6]
```
#### Example 3:
```
Input: [1, 5, 6, 4, 3, 2]
Output: [1, 2, 3, 4, 5, 6]
```
